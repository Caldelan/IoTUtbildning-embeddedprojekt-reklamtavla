##Reklamtavla
Kod för att slumpmässigt visa meddelanden på en simulerad arduino
simulatorn -> https://wokwi.com/projects/416241646559459329

vilken 'kund' som visar meddelandet slumpas fram, aldrig samma kund två gånger på rad och större chans att 'kunden's meddelande visas är beroende på hur mycket de 'betalat'

några meddelanden ska bara visas, andra blinka och några scrollas.
